## Additional Notes

### DPDP Rules 2025
The DPDP Rules 2025 were notified on November 13, 2025, with phased implementation. Key points:
- **Phase 1** (Nov 13, 2025): Board establishment, definitions, core administrative functions
- **Phase 2** (Nov 13, 2026): Consent Manager registration
- **Phase 3** (Mar 13, 2027): Full compliance — notice, consent, data principal rights, security safeguards, breach notification (72 hours per Rule 21), cross-border transfers, SDF obligations

The code references some Rules-level obligations (e.g., 72-hour breach notification) as if they are in the Act itself. This is technically inaccurate — the Act establishes the obligation, the Rules specify the timeline.
